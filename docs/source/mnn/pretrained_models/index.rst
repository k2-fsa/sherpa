.. _sherpa-mnn-pre-trained-models:

Pre-trained models
==================

In this section, we describe how to download and use
available pre-trained models.

.. hint::

   Currently, `sherpa-mnn`_ only supports zipformer-transucer-based models.
   Please refer to https://k2-fsa.github.io/sherpa/ncnn/pretrained_models/index.html
   to use other models.

.. hint::

  Please install `git-lfs <https://git-lfs.com/>`_ before you continue.
