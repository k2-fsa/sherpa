Huggingface space
=================

You can try `Cohere Transcribe`_ with `sherpa-onnx`_ with the following huggingface space

  `<https://huggingface.co/spaces/k2-fsa/automatic-speech-recognition>`_


.. hint::

   You don't need to install anything. All you need is a browser.

   You can even run it on your phone or tablet.

.. figure:: ./pic/cohere-hf.jpg
   :alt: screenshot of hf space for Cohere Transcribe
   :align: center
   :width: 600

   Try `Cohere Transcribe`_ in our Huggingface space with `sherpa-onnx`_

.. note::

   You need to also select the language.
