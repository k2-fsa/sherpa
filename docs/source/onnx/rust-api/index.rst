Rust API
========

In this section, we describe how to use the `Rust <https://github.com/k2-fsa/sherpa-onnx/tree/master/rust-api-examples>`_ API of ``sherpa-onnx``.

The Rust crate lives in the `sherpa-onnx repository <https://github.com/k2-fsa/sherpa-onnx/tree/master/sherpa-onnx/rust>`_ and is published on crates.io as
`sherpa-onnx <https://crates.io/crates/sherpa-onnx>`_. API documentation is
available at `docs.rs <https://docs.rs/sherpa-onnx>`_.

We also provide a collection of runnable examples in
`rust-api-examples <https://github.com/k2-fsa/sherpa-onnx/tree/master/rust-api-examples>`_.
