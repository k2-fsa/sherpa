Windows
=======

This page explains how to install sherpa-onnx_ on Windows.

.. hint::

   MinGW is not supported and will not work with sherpa-onnx_.

   Please install ``Visual Studio 2022`` or a newer version before proceeding.
   Earlier versions of ``Visual Studio`` may not be compatible.
