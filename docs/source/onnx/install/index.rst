Installation
============

In this section, we describe how to install `sherpa-onnx`_ on various platforms.

**Requirements**:

  - ``CMake >= 3.13``
  - A compiler that supports at least ``C++14``

.. hint::

   You can use ``pip install cmake`` to install the latest cmake.
