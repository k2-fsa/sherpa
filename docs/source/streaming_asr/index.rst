Streaming ASR
=============

This page describes how to use `sherpa`_ for streaming ASR.

Currently, implemented streaming ASR models include:
`Emformer`_ `transducer`_ (i.e., Emformer-T)
and `ConvEmformer`_ `transducer`_ (i.e., ConvEmformer-T).
