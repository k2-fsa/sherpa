Node-Addon Examples
===================

This section documents the runnable JavaScript examples for the
``sherpa-onnx-node`` native addon. Each page below explains what the example
demonstrates, shows the full source code, and provides step-by-step
instructions for running it.

The source files are also available in the
`nodejs-addon-examples <https://github.com/k2-fsa/sherpa-onnx/tree/master/nodejs-addon-examples>`_
directory of the sherpa-onnx repository.
