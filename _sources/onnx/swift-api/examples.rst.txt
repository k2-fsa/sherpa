Examples
========

Please see `<https://github.com/k2-fsa/sherpa-onnx/tree/master/swift-api-examples>`_


Each example has a corresponding shell script. Please use the shell script to run it.


For instance,

  - to run the text-to-speech example, please use `<https://github.com/k2-fsa/sherpa-onnx/blob/master/swift-api-examples/run-tts.sh>`_
  - to run the speech-to-text example, please use `<https://github.com/k2-fsa/sherpa-onnx/blob/master/swift-api-examples/run-decode-file.sh>`_
