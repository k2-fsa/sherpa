SenseVoice

This section describes how to use models from https://github.com/FunAudioLLM/SenseVoice.

A single model from SenseVoice supports the following languages

  • Chinese (Mandarin, 普通话)

  • Cantonese (粤语, 广东话)

  • English

  • Japanese

  • Korean
