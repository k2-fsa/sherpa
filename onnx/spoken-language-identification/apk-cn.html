APKs for spoken language identification

This page lists the spoken language identification APKs for sherpa-onnx, one of the deployment frameworks of the Next-gen Kaldi project.
The name of an APK has the following rule:
  • sherpa-onnx-{version}-{arch}-slid-{model}.apk
where
  • version: It specifies the current version, e.g., 1.9.21
  • arch: The architecture targeted by this APK, e.g., arm64-v8a, armeabi-v7a, x86_64, x86
  • model: The name of the model used in the APK


You can download all supported models from https://github.com/k2-fsa/sherpa-onnx/releases/tag/asr-models

Please see https://k2-fsa.github.io/sherpa/onnx/spolken-language-identification/ for more information.

Note about the license The code of Next-gen Kaldi is using Apache-2.0 license. However, we support models from different frameworks. Please check the license of your selected model.
