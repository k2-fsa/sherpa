Flutter Apps for Linux about streaming speech recognition

This page lists the streaming speech recognition Flutter Apps for sherpa-onnx, one of the deployment frameworks of the Next-gen Kaldi project.
The name of an App has the following rule:
  • sherpa-onnx-{version}-{os}-{arch}-asr-{lang}-{model}
where
  • version: It specifies the current version, e.g., 1.12.14
  • os: It specifies the OS, e.g., osx for macOS, win for Windows, linux for Linux
  • arch: The architecture targeted by this App
  • lang: The lang of the model used in the App, e.g., en for English, zh for Chinese
  • model: The name of the model used in the App

You can download all supported models from https://github.com/k2-fsa/sherpa-onnx/releases/tag/asr-models

Source code for building the APP is publicly available at https://github.com/k2-fsa/sherpa-onnx/tree/master/flutter-examples/streaming_asr
