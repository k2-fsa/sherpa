Hint

If you don’t want to build sherpa-onnx from source for Windows arm64, please continue reading.

This section describes how to install prebuilt sherpa-onnx binaries on Windows arm64.

Please visit

https://github.com/k2-fsa/sherpa-onnx/releases/tag/v1.12.24

to download pre-compiled executables and libraries for Windows arm64.
