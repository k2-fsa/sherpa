Tauri desktop apps for non-streaming speech recognition from microphone

This page lists the non-streaming speech recognition Tauri desktop apps for sherpa-onnx, one of the deployment frameworks of the Next-gen Kaldi project.
The apps listed on this page use live microphone input with VAD to detect speech segments, then recognize each segment with an offline ASR model. They display wall-clock timestamps, support recording playback, SRT export, and segment WAV export.
The source code of the app is at https://github.com/k2-fsa/sherpa-onnx/tree/master/tauri-examples/non-streaming-speech-recognition-from-microphone
You can find pre-built apps at https://huggingface.co/csukuangfj2/tauri-app

The name of an archive has the following rule:
  • vad-asr-mic-{version}-{lang}-{model}-{platform}.{ext}
where
  • version: It specifies the current version, e.g., 1.12.39
  • lang: The language of the model used in the app, e.g., en for English, zh for Chinese
  • model: The name of the model used in the app
  • platform: The target platform, e.g., linux-x64, linux-aarch64, universal.app (macOS), windows-x64
  • ext: .tar.gz for Linux, .tar.bz2 for macOS, .zip for Windows

You can download all supported models from https://github.com/k2-fsa/sherpa-onnx/releases/tag/asr-models

Note about the license The code of Next-gen Kaldi is using Apache-2.0 license. However, we support models from different frameworks. Please check the license of your selected model.
